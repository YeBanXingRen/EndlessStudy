#### 段落格式

换行：是使用两个以上的空格加上回车

test  

test

#### 字体

斜体：*包围、_包围

*斜体*



粗体：**包围、__包围

**粗体**



粗斜体：***包围、___包围

***粗斜体***



删除线：~~包围

~~删除线~~  



下划线：标签<u>包围

<u>下划线</u>  



脚注：中括号内用^开始

test[^脚注]

[^脚注]:这是一个脚注





#### 分割线

三个*或三个- 本行不能有其他东西

***



#### 列表

用*或+或-开头 再加一个空格

* 第一项
* 第二项

- 第三项

* 区块

  在>后紧跟一个空格

  > 区块
  >
  > > 嵌套区块

#### 代码

一行代码用`引起来

`printf("hello world");`

代码段用三个`引起来 且可以设置代码类型

```java
private static void main(){
    
}
```
