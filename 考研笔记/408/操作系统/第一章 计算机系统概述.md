# 第一章 计算机系统概述

操作系统的基本概念

```
操作系统的概念
	什么是操作系统：能合理分配计算机资源给到所有的应用程序，能调度计算机软件规范运行的系统软件。
操作系统的特征
	并发：能处理和调度多个程序在同一时间间隔内执行，并发是通过分时实现的，微观上依然是串行执行，只不过多道程序交替执行。
	共享：系统中的资源可供内存中多个并发执行的进程共同使用。
		互斥共享方式：规定一段时间内只允许一个进程访问该资源。该资源被称为临界资源。打印机。
		同时访问方式：允许多个进程同时访问一个资源。
		并发和共享是操作系统最基本的特征。
	虚拟：把一个物理上的实体变成若干逻辑上的对应物。虚拟处理器，虚拟内存，虚拟外部设备。
	异步：异步使得操作系统运行在一个随机的环境下，可能导致进程产生与时间相关的错误。
操作系统的目标和功能
	目标：为多道程序提供良好的运行环境。
	功能：处理机管理、存储器管理、设备管理、文件管理、接口。
		对处理机的管理可以归结为对进程的管理，进程控制、进程同步、进程通信、死锁处理、处理机调度。
		存储器管理包括内存分配和回收、地址映射、内存保护和共享、内存扩充
		文件管理：文件存储空间、目录管理、文件读写管理和保护
		设备管理：缓冲管理、设备分配、设备处理、虚拟设备
		接口：
			命令接口
				联机命令接口，向系统提出要求，系统自己执行程序满足要求
				脱机命令接口，向系统提供执行过程，系统按照执行过程执行。
			程序接口
				交由应用程序使用的接口。应用程序提出要求，系统满足程序的需求。
```
