# 计算机组成原理第四章

## 指令系统

```
指令格式
	指令基本格式
		操作码+地址码
		零地址、一地址、二地址、三地址、四地址
		其中一地址有两种情况，即只有一个操作数的指令和一个操作数与ACC运算的指令
	定长操作码指令格式
		操作码字段长度固定
	扩展操作码指令格式
		操作码字段长度不固定且短码不能与长码的前面部分相同
	指令操作类型
		数据传送、算数逻辑运算、移位、转移、输入输出操作
寻址方式
	指令寻址和数据寻址
		指令寻址分为顺序寻址和跳跃寻址，其本质都是修改pc，下一条指令地址一定由pc给出
		数据寻址，寻址特征字段
	常见的数据寻址方式
		隐含寻址
		立即数寻址
		直接寻址
		间接寻址
		寄存器寻址
		寄存器间接寻址
		相对寻址
		基址寻址
		变址寻址
		堆栈寻址

程序的机器级代码表示

CISC和RISC
```
