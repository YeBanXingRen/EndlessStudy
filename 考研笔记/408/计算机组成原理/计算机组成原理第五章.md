# 计算机组成原理第五章

## CPU

```
CPU的功能和基本结构
	CPU的功能：
		指令控制、操作控制、时间控制、数据加工、中断处理
	基本结构：
		运算器、控制器
		运算器核心部件为ALU，其余部件如暂存器、累加器ACC、通用寄存器组、程序状态字寄存器PSW、移位器、计数器CT。
		控制器基本功能是执行指令，由程序计数器PC、地址寄存器MAR、数据寄存器MDR、指令寄存器IR、指令译码器、时序系统和位操作信号发生器。
		
		
指令执行过程
	指令周期
		指令周期包含多个机器周期，每个机器周期包含多个时钟周期，机器周期可以是定长也可以是不定长。一个指令周期可能包含四种机械周期，即取指周期、间址周期、执行周期、中断周期。为了区别不同的周期，为这四种周期设置了四个标志触发器，并以状态1表示有效。
	指令周期的数据流
		取指周期、间址周期、执行周期、中断周期
	指令执行方案
		单指令周期、多指令周期、流水线方案
		
		
数据通路的功能和基本结构（cpu内部总线）
	数据通路的功能：实现CPU内部的运算器与寄存器及寄存器之间的数据交换
	数据通路的基本结构
		CPU内部单总线方式
		CPU内部三总线方式
		专用数据通路方式
	寄存器之间的数据传送
	主存与CPU之间的数据传送
	执行算术逻辑运算
	
	
控制器的功能和工作原理
	控制器的功能：
		从主存取指令并指出下一条指令的位置
		对指令进行译码，产生相应的控制信号
		控制CPU、主存、输入输出设备之间的信息流动
	硬布线控制器
		其基本原理是根据指令的要求、当前的时序及外部内部状态，按时间顺序发送一系列微操作控制信号，它由复杂的组合逻辑门电路和触发器构成，又称为组合逻辑控制器。
		
	微程序控制器
异常和中断
指令流水线
多处理器基本概念
```
