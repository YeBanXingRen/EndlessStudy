# 计算机组成原理第二章

## 数据表示和运算

```
数制和编码：
	2，8，10，16进制的相互转换，10->2，8，16 除k取余法和乘k取整法
							2，8，16->10 各位数乘权重
	BCD码：8421码，余3码和2421码
			余3码是在8421码的基础上每一个数都加上(0011)
			2421码小于5的最高位都为0

定点数的编码表示：
	原码，补码，反码，移码。
	前三种码制正数都为真值。移码的正数与负数计算方式统一，都为真值+2的n次方
	原码转补码：各位取反，末尾加一
	原码转反码：各位取反
	
移位运算：
	算数移位，逻辑移位，循环移位
	算数移位：算数移位不管如何移动，其结果都是在真值上添0。原码，补码，反码的正数算术移位都是填0，负数的原码添0，反码添1，补码左移添0右移添1。
	逻辑移位：多余位丢弃，一律补0
	循环移位：带/不带进位表示的循环左/右移。
	
定点数的加减乘除四则运算：
	加减：补码运算方法	A+B的补码=A的补码+B的补码
					A-B的补码=A的补码+[-B]的补码
	乘法：原码一位乘，补码一位乘
	除法：原码除法，补码除法
	
c语言中的整数类型及类型转换：

数据的存储和排列：
	大小端存储
	数据按边界对齐

浮点数的表示与运算：
	浮点数的表示格式
	IEEE754规范
	浮点数的加减运算，规格化处理。
```
