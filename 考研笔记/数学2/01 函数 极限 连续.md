### 函数

函数的概念

```
了解为主
会求函数表达式

Df = X	定义域
Rf <= Y	Rf为值域 Y为所有y的集合
单射	当Rf = Y时，称为单射。单射 <=> 函数单调 <=> 函数有反函数 <=>图像与任意常数函数相交只有一个交点
一一映射	一个x有且仅有一个y与之对应
```

函数的几何特性

```
奇偶性
	定义	奇函数f(x) = -f(-x) <=> f(x) + f(-x) = 0;偶函数f(x) = f(-x) <=> f(x) - f(-x) = 0;
	图像特点	奇函数关于原点对称 f(0) = 0;	偶函数关于y轴对称;在求定积分时会使用到"奇0偶倍";
	常见奇偶函数	奇函数 f(x)-f(-x),ln(x-根号下1-x*x);偶函数 f(x)+f(-x)
	奇偶性的运算
		加减:奇+/-奇=奇, 偶+/-偶=偶, 奇+/-偶=非奇非偶
		乘除:奇*/奇=偶, 偶*/偶=偶, 奇*/偶=奇	(奇看作负数，偶看作正数)
		复合:奇复合奇为奇 其他全为偶
周期性
	定义	f(x) = f(x+T) T为周期
	常见周期函数
		y=sin^2(x) T=PI
有界性
	几何意义
	概念
单调性
	定义
	判断
```

极限的运算法则

```
基本四则运算及其推广
	极限可拆的原则: 在极限存在法则中没有存在性未知的项
	极限存在的运算法则： 存在 (+/-) (存在/不存在) = (存在/不存在); 不存在 (+/-) 不存在 = 未知; 乘除运算必须都存在才能得到存在，否则存在性未知
	[f(x)/g(x)]的极限为A时，若g(x)=0;则f(x)=0; 若f(x)=0且A!=0，则g(x)=0
	[f(x)*g(x)]的极限为A时，若f(x)=0，则g(x)=∞
```
