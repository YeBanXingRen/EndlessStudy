极限存在的判别方法

```
夹逼定理
	数列和函数都可以使用此方法
	会运用到放缩的思想
		分式放缩-->分母放大到最大值、缩小到最小值
		整式放缩-->放大一般放大到最大因式的n倍，即n*max；缩小一般缩小到最大因式，即max。（我的理解）原因：放缩的目的是为了能正常使用到夹逼定理，而夹逼定理需要放缩后的两个式子在n->∞的时候的极限相同，而如此放缩之后，放大的极限和缩小的极限之间只相差一个常数倍数，且max是趋近于无穷的，所以这个常数并不影响两个极限都趋近于∞。
单调有界数列必有极限
	常用来证明数列的极限存在。要证明数列极限存在需要同时满足两个条件，即单调和有界。可以更加细分为两种情况：单增有上界，单减有下界。
		单调的证明方法：1.相减/相除。2.数学归纳法
		有界的证明方法：1.利用常见不等式。2.数学归纳法
		此证明题中数学归纳法的一般步骤：1.设第k项满足某种条件，推出第k+1项也满足此条件。2.第一项也满足此条件-->xn全都满足-->单调/有界。
		常见不等式：sinx<x<tanx	x/1+x<ln(1+x)<x	a+b>2根号下ab
```

无穷小和无穷大

```
定义
	都是函数
无穷小与无穷大之间的关系
	互为倒数，特例：常数函数f(x)=0;
无穷小的运算
	无穷小+无穷小=无穷小
	无穷小*无穷小=无穷小
	有界函数*无穷小=无穷小
无穷小量之间的比较以及利用等价无穷小来计算极限
	常用等价无穷小
		x ~ sinx ~ tanx ~ arcsinx ~ arctanx ~ (e^x)-1 ~ ln(1+x)
		1-cosx ~ x-ln(1+x) ~ (1/2)x^2	[(1+x)^a]-1 ~ ax
		x-sinx ~ (1/6)x^3	x-arcsinx ~ -(1/6)x^3
		tanx-x ~ (1/3)x^3	arctanx-x ~ -(1/3)x^3
	较特殊
		(a^x)-1 ~ xlna	ln(1+f(x)) ~ f(x)	(这两个是 (e^x)-1 和 ln(1+x) 的推广)
		带x^a的分式中使用抓大头
		变限积分中的等价
极限与无穷小之间的关系
	若f(x)的极限等于A，则f(x)=A+α(x)，其中α(x)的极限等于0
等价无穷小的性质
	a等价于b <=> a=b+o(b)
```

### 连续

定义

```
连续的定义：左极限等于右极限等于这点的函数值
左连续/右连续：左极限/右极限存在且等于这点的函数值
连续的充要条件：函数在这点上即左连续又右连续（本质还是定义）
区间连续：开区间连续->区间内每一点都连续；闭区间连续->区间内每一点连续且临界点单侧连续
```

有关连续的方法和结论

```
基本初等函数在定义域内连续
连续函数的和差积商仍连续
连续函数的复合函数、反函数仍连续
初等函数在定义区间内连续
```

间断点

```
第一类间断点：左右极限都存在
	可去间断点->左右极限相等但不等于该点函数值
	跳跃间断点->左右极限不相等
第二类间断点：左右极限都不存在
	无穷间断点->左右极限中至少有一个等于无穷
	振荡间断点->非以上三种	例如sin(1/x)
```

闭区间连续函数的性质

```
最值定理
	闭区间上的连续函数一定有最大值和最小值
有界性定理
	闭区间上的连续函数在区间内一定有界
介值定理
	闭区间上的连续函数必然能取到介于最大值和最小值之间的任何值
零点定理
	设函数在闭区间上连续，且端点上的函数值异号，则一定存在函数值等于零的点
```
