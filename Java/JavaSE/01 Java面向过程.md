### Java概述

```
Java开发环境搭建：
	JDK、JRE、JVM
	修改全局变量
Java的平台无关性：
	Java的平台无关性是通过JVM的跨平台实现的
	.class文件、.java文件
垃圾回收机制
```

### 变量

```
按变量声明所处的位置分类：
	局部变量、成员变量
按数据类型分类：
	基本数据类型：
		8种；整型：int short lang byte 浮点型：float double 字符型：char 布尔型 boolean
	引用数据类型：
		数组、对象、包装类及其他非基本数据类型
数据类型转换:
	强制类型转换、自动类型转换
```

### 运算符

```
算数运算符：
	+、-、*、/
关系运算符：
	>(大于)、>=(大于等于)、==(等于)、<(小于)、<=(小于等于)、!=(不等于)
逻辑运算符：
	逻辑与(&)、逻辑或(|)、逻辑非(!)、短路与(&&)、短路或(||)
条件运算符:
	(A>B)?C:D -->真输出C、假输出D
赋值运算符
	=、+=、-=
```

### 程序控制结构

```
判断结构：
	if else、switch case
循环结构
	for、while、do while
跳转结构
	break、continue
```
